# Worker Servisi

YouTube Otomasyon Sistemi için arkaplan işlerini yürüten worker servisi.

## Teknolojiler

- Node.js
- Bull.js (Redis tabanlı iş kuyruğu)
- Sequelize ORM
- Google API İstemcileri
- AI API İstemcileri (OpenAI, Claude)

## İş Kuyrukları

- İçerik Planlama
- İçerik Üretimi 
- Görsel Üretimi
- YouTube Yükleme
- Performans Analizi
- Rakip Analizi
- Bildirim Gönderimi
